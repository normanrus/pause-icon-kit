# Pause Icon Kit 🛠

A Geode mod for Geometry Dash that allows you to access the Icon Kit (Garage) directly from the in-game pause menu. Change your icons, ships, waves, or colors mid-level without losing your current progress!

## Experimental Mode

Due to how Geometry Dash handles scenes, entering the garage during a live level triggers **Experimental Mode**. It has a couple of unique quirks that you should be aware of:

1. **Immediate Action:** As soon as you leave the garage and click **"LET'S GO!"**, the level will resume instantly. Make sure you are ready to jump right back into the gameplay!
2. **Menu Reset:** If you decide to quit the level after visiting the garage, the game will reset you back to the Main Menu instead of the level select screen.

## Features

* **Quick Access:** A new Garage button added smoothly into the right side menu of the `PauseLayer`.
* **Live Updates:** Your icons, colors, and death effects update immediately upon returning to the level.
* **Geode Native:** Safe, lightweight, and fully compatible with the Geode ecosystem.
